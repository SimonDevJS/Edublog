# Objetivo 1.3 — Análisis de Datos en Excel

Como parte del **objetivo 1.3** de la asignatura Introducción a la Informática, en esta actividad se utilizó **Microsoft Excel** para emplear herramientas de análisis de datos, funciones y fórmulas.

Este ejercicio tuvo como objetivo mejorar las habilidades en la gestión de hojas de cálculo, la estructuración de información y el análisis de resultados numéricos.

## Pasos realizados

1. Se construyeron tablas de datos con valores numéricos.
2. Se utilizaron funciones aritméticas simples y funciones como **SUMA**, **PROMEDIO**, **MAX** y **MIN**.
3. Se utilizaron formatos condicionales para mejorar la visualización de los resultados.
4. Se crearon gráficos simples para ilustrar visualmente los datos.

## Conclusión

Este ejercicio consolidó el empleo de **Microsoft Excel** como una herramienta esencial para la gestión y análisis de datos.
Para resolver problemas en situaciones académicas y laborales, es crucial dominar estas funciones fundamentales.
