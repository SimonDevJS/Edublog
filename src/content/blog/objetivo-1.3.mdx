# Objetivo 1.3 — Análisis de Datos en Excel

En esta actividad se trabajó con **Microsoft Excel**, aplicando fórmulas, funciones y herramientas de análisis de datos como parte del **Objetivo 1.3** de la asignatura _Introducción a la Informática_.

A través de este ejercicio se buscó fortalecer las competencias en el manejo de hojas de cálculo, la organización de información y la interpretación de resultados numéricos.

## Pasos realizados

1. Se construyeron tablas de datos con valores numéricos.
2. Se aplicaron fórmulas aritméticas básicas y funciones como **SUMA**, **PROMEDIO**, **MAX** y **MIN**.
3. Se utilizaron formatos condicionales para mejorar la visualización de los resultados.
4. Se elaboraron gráficos sencillos para representar los datos de forma visual.

## Conclusión

Esta práctica permitió afianzar el uso de **Microsoft Excel** como herramienta fundamental en la gestión y análisis de información.  
El dominio de estas funciones básicas resulta esencial para la resolución de problemas en contextos académicos y profesionales.
