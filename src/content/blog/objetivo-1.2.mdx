# Objetivo 1.2 — Componentes del Computador

En esta actividad se presenta un resumen visual sobre los **principales componentes del computador**, tanto físicos como lógicos, y su importancia en el funcionamiento del sistema.

## Introducción

Cuando usamos un ordenador, la mayoría de las personas no son conscientes de las partes que lo componen.  
Existen elementos **visibles**, como el teclado o la pantalla, y otros **no visibles**, como el sistema operativo o las aplicaciones.  
El **hardware** y el **software** son las dos partes fundamentales que permiten que un computador funcione correctamente.

## Contenido Destacado

- **CPU (Procesador):** el “cerebro” del computador, encargado de coordinar y ejecutar instrucciones.
- **Memoria RAM:** almacena datos temporalmente; influye directamente en la velocidad del sistema.
- **Disco Duro:** almacena la información de manera permanente (HDD o SSD).
- **Sistema Operativo:** conecta al usuario con el hardware. Ejemplos: Windows, Linux, macOS.
- **Aplicaciones:** programas creados para tareas específicas como escribir, navegar o editar imágenes.

## Conclusión

El hardware y el software **trabajan juntos como una unidad inseparable**.  
Comprender su relación nos ayuda a **aprovechar mejor la tecnología** y mantener nuestros equipos en buen estado.
