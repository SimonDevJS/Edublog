# Objetivo 1.2 — Componentes del Computador

En esta actividad se proporciona un resumen visual de los **componentes más importantes del ordenador**, tanto físicos como lógicos, y la relevancia que tienen en el desempeño del sistema.

## Introducción

Cuando usamos un ordenador, la mayoría de las personas no son conscientes de las partes que lo componen.
Existen elementos **visibles**, como el teclado o la pantalla, y otros **no visibles**, como el sistema operativo o las aplicaciones.
El **hardware** y el **software** son las dos partes fundamentales que permiten que un computador funcione correctamente.

## Contenido Destacado

- **CPU (Procesador):** El "cerebro" del ordenador que tiene la responsabilidad de organizar y llevar a cabo instrucciones.
- **Memoria RAM:** Guarda datos de manera temporal; afecta directamente la velocidad del sistema.
- **Disco Duro:** Guarda la información de forma permanente (HDD o SSD).
- **Sistema Operativo:** Vincula al usuario con el hardware. Por ejemplo: Linux, Windows y macOS.
- **Aplicaciones:** Software diseñado para realizar tareas concretas, como editar imágenes, navegar o redactar.

## Conclusión

El hardware y el software **funcionan en conjunto como una unidad inseparable**.
Entender su relación nos permite aprovechar mejor la tecnología y conservar nuestros equipos en óptimas condiciones.
